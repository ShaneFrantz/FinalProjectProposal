Application Overview

This application will be a fully playable clone of the popular NYTimes game "Wordle". Wordle is a word based game in which the player must guess a five letter word in six guesses by typing in their own five letter words. After each guess, they will learn which letters of their guess word are in the actual word, and if they are already in the correct position or not.

Technology Overview

Technologies used will be:

  • HTML
  • CSS
  • JavaScript
  • SQL (database for the words)
  • Azure (for hosting the website)

Feature List

  • Database with a random word selected upon each load to be the puzzle word
  • A field for the user to guess and see which letters they've correctly found
  • Showing the user the correct word when the round is over
  • Not allowing the user to type an invalid word
  • Handling all game logic correctly
  • A UI to show previous guesses
  • Prevent the user from typing any input longer than 5 letters
